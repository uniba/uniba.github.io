# 入社時・退職時手続き

スタッフの入社時および退職時のやることリスト。

## 入社時手続き
* 雇用契約書の締結
  - 2部作成、双方押印の上保管

* 提出書類
  - 履歴書
  - 年金手帳 (年金番号)
  - 前職退職時の源泉徴収票 (当年中の前職所得がある場合のみ)
  - 住民票 (住宅補助制度適用の場合)
  - 個人番号を確認できる書類の提示
  - 口座振替依頼書

* 各種サービスのアカウント作成
  - GSuite
  - Slack
  - Makeleaps
  - github
  - shukei.io
  - Trello
  - figma

* サブスクリプションソフトウェアのライセンス割り当て
  - Sketch
  - Adobe CC
  - Office365

* 席・棚・カードキー・PC貸与
  - 固定席
  - 個人棚（ボックス・ステッカー）
  - カードキー
  - (PC)

* ユニバの働き方についての説明

## 退職時手続き
* 貸与物の返却
  - 固定席
  - 個人棚
  - カードキー
  - (PC)
  - (サブスクリプションソフトウェアのライセンス割り当て)

* 退職に伴って発行される書類
  * 源泉徴収票
  * 雇用保険被保険者証
  * 離職票
