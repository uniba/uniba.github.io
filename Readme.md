# uniba.jp

Corporate site of Uniba Inc.

## Installation

Clone repository and solve dependencies.

    $ git clone git@github.com:uniba/uniba.github.io.git
    $ cd uniba.github.io
    $ bundle install --path vendor/bundle

## Workflow

To make preview server.

    $ make server

## Deployment

    $ make deploy
